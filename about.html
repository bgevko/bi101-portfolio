Who am I?

My name is Bogdan Gevko and I a Computer Science student at Oregon State University.

What is this?

This website was used to satisfy a final project requirement in Biology 101 at Oregon State University. It showcases a body of work that proves that I achieved course mastery.

Why I chose this format

There are many benefits to creating a website over a traditional presentation.

  • I can look back on my work in the future, which was one of the requirements for the final.

  • I got to improve my web development skills.

  • I learned a new web framework, Jekyll.

  • This is my first website and I had a lot of fun making it.

How does this website represent mastery?

Each section is divided into three subsections: Concepts, Skills, and Connections. The uploaded images are media assignments that I completed throughout the course. The purpose of each section is to explain how each media piece meets each respective requirement.
